История изминений G740Viewer

-----------------------------------
Версия от 2019-09-15
-----------------------------------
Модальная экранная форма
	Исправлена ошибка с продолжением выполнения списка запросов при закрытии модальной экранной формы

Панель FIELDS
	Для числовых полей сделан отступ справа для того, чтобы кнопочки очистки не перекрывала содержимое поля

-----------------------------------
Версия от 2019-09-12
-----------------------------------
Модальная экранная форма
	Закрытие последней модальной экранной формы, по аналогии с закрытием обычной формы, разворачивает главное меню системы.

Панель FIELDS
	Увеличена минимальная ширина поля на 8px для того, чтобы кнопочки очистки не перекрывала содержимое поля

-----------------------------------
Версия от 2019-09-11
-----------------------------------
Обработка ошибки при выполнении запроса save
	В диалоге сообщения об ошибке, при выполнении запроса save, появилась дополнительная кнопочка - отменить несохраненные изменения

-----------------------------------
Версия от 2019-09-10
-----------------------------------
Справочник
	При вызове диалога справочника, если заполнен фильтр, строка пустого значения перемещается в конец списка, дабы верхнее значение более всего соответствовало введенному фильтру

-----------------------------------
Версия от 2019-09-09
-----------------------------------
Панель FIELDS
	Уменьшена ширина кнопочки очистки поля с 23px до 11px, содержимое коротких полей теперь остается видимым при наведении мышки

-----------------------------------
Версия от 2019-09-05
-----------------------------------
Панель TreeMenu
	- Стало можно из главного меню приложения выполнять произвольные запросы.
		Для этого запрос надо объявить в главной экранной форме приложения, а в меню указать его имя, вместо имени запускаемой экранной формы.
request httpput
	- Теперь в параметре ext можно указывать любые расширения, разделенные ';' например: jpg;png;gif;doc;dxf;ini

-----------------------------------
Версия от 2019-08-28
-----------------------------------
Панель GRID
	Дополнительный параметр поля align = left|right|center

-----------------------------------
Версия от 2019-08-23
-----------------------------------
Имена (caption) запросов
	Переименованы caption запросов append, undo, delete - текст короче, добавлено сочетание клавиш
Размеры иконок в кнопках
	В кнопках панелей размер иконок всегда medium

-----------------------------------
Версия от 2019-08-21
-----------------------------------
Панель HTMLLIST
	Нестандартное вычисление макроподстановки в шаблоне посредством js_tpvalue.
		теперь в js_tpvalue передается имя макроподстановки;
		теперь если js_tpvalue не знает имя макроса, она должна вернуть пустое значение, и далее пойдет стандартная обработка;
Панель GRID
	Поле type=html - отображаемый в ячейке html текст
		реализован класс table для таблиц внутри такого поля, растягивается на всю ширину:
			<table class="table"> ... </table>
		вариант с подсветкой текущеей строки
			<table class="table hover"> ... </table>
