Версия от 2019-09-29
-----------------------------------
Отображение полей в режиме captionup=1
	Если поле не stretch и заголовок шире длины поля, то поле не растягивается по ширине заголовка

Обработка запросов главной формой - контейнером экранных форм
	Если запроса нет в текущей форме, и он объявлен в главной форме, то запрос переадресуется на главную форму

Версия от 2019-09-26
-----------------------------------
Добавлена панель attribute

Переделана поддержка action полями анкет
	поле с action стало нередактируемым
	двойной клик по полю с action вызывает выполнение action, указатель мыши - pointer
	при вызове модального окна экранной формы, по закрытии, не терется фокус ввода поля
	если поле readonly, action не вызывается

-----------------------------------
Версия от 2019-09-25
-----------------------------------
Изменен запрос copy
	добавлено mode= first|last|after|before
	параметр <имя поля>=значение - подменить значение поля значением из параметра
	параметр <имя поля>.rename=1 - добавить к значению поля слово <копия>
	
Параметр запроса lock
	Если у запроса стоит параметр lock=1 то при пока он выполняется, отображается экран блокировки с бегущим колесиком. Пока работает только для одиночных запросов, вызываемых по кнопке.

Версия от 2019-08-21
-----------------------------------
Панель HTMLLIST
	Нестандартное вычисление макроподстановки в шаблоне посредством js_tpvalue.
		теперь в js_tpvalue передается имя макроподстановки;
		теперь если js_tpvalue не знает имя макроса, она должна вернуть пустое значение, и далее пойдет стандартная обработка;
Панель GRID
	Поле type=html - отображаемый в ячейке html текст
		реализован класс table для таблиц внутри такого поля, растягивается на всю ширину:
			<table class="table"> ... </table>
		вариант с подсветкой текущеей строки
			<table class="table hover"> ... </table>
